MOVIE API

This app is a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Endpoint HTTP Method Description
/movies GET Returns all movies.
/users GET Returns all users.
/movies/:Title GET Returns a movie based on the provided title.
/movies/genres/:genreName GET Returns movies based on the provided genre name.
/movies/directors/:directorName GET Returns data from a director based on the provided name.
/users POST Allows new users to register.
/users/:Username PUT Allows users to update their information (username, password, email, birthday).
/users/:Username/movies/:MovieID POST Allows users to add movies to their favorites list.
/users/:Username/movies/:MovieID DELETE Allows users to remove a movie from their favorites list.
/users/:Username DELETE Allows users to delete their registration.
/ GET Returns a simple text message.
Static Files N/A Serves static files from the "public" folder.
Error Handling N/A Logs all errors to the terminal.
